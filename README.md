# PROSPECT

Браузерная игра о жизни и карьере профессионального спортсмена.

## Текущий этап

Версия 0.11 — завершён школьный вертикальный срез:

- полноценный школьный сезон;
- матчи за атаку и защиту;
- команда, тренеры и динамический depth chart;
- здоровье, тренировки и развитие;
- постоянные NPC, отношения и память;
- реалистичный рекрутинг, визиты и предложения;
- формальное подписание стипендии или preferred walk-on;
- выпускное межсезонье и прибытие в колледж;
- реальная позиционная комната и проверка обещаний штаба;
- IndexedDB, миграции, экспорт и резервные сохранения.

## Команды

npm install
npm run dev

npm run build
