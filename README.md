# PROSPECT

Браузерная игра о жизни и карьере профессионального спортсмена.

## Текущий этап

Патч 1 — технический фундамент:

- React + TypeScript + Vite;
- мобильный интерфейс;
- PWA и офлайн-кэш;
- несколько карьер;
- IndexedDB;
- автосохранения и резервные снимки;
- импорт и экспорт карьер;
- версионирование схемы;
- seeded random;
- динамическая загрузка American Football;
- базовые тесты.

## Команды

```bash
npm install
npm run dev
npm run test
npm run build
```
