# PROSPECT

Браузерная игра о жизни и карьере профессионального спортсмена.

## Текущий этап

Версия 0.28.0:

- Профиль, Дом и Команда в нижней навигации;
- сезон, матчи, таблица, обзор, рекрутинг, лента и рейтинг в боковой панели;
- статистика и решения игрока в профиле;
- полноценные страницы команд: состав, штаб, система и ресурсы;
- компактные строки сохранений и рабочее удаление;
- числовые эффекты тренировок, архетипов и характера;
- лента мира без служебных описаний и нулевых сводок.

- единый шрифт Inter и фиксированная типографическая шкала;
- единый мобильный отступ 20 px и сетка 4 px;
- радиусы обычных карточек ограничены 16 px;
- плоская иерархия без декоративных вложенных карточек;
- только один активный раздел навигации;
- повышенная плотность профиля, команды, дня, мира и сохранений;

## Команды

```bash
npm install
npm run dev
npm run test
npm run build
```
