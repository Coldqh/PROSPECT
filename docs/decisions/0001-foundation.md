# ADR-0001: фундамент PROSPECT

## Решения

- Один репозиторий и одно общее жизненное ядро.
- Активный спорт фиксируется внутри карьеры.
- Football загружается через динамический импорт.
- Basketball пока существует только как недоступный дескриптор.
- Для GitHub Pages используется HashRouter.
- IndexedDB — единственное постоянное хранилище карьер.
- Каждое сохранение имеет schemaVersion, revision и checksum.
- Math.random запрещён в игровой логике.
