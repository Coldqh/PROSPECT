# Relationships and contextual life events

PROSPECT uses one visible relationship value per NPC, from -100 to +100.

The number answers one question: how the person currently relates to the hero. It does not replace personality, goals, fears, current pressure or memory.

## Permanent people

The high-school slice creates seven stable people from the world seed:

- guardian;
- head coach;
- position coach;
- direct positional rival;
- senior teammate;
- school counselor;
- local reporter.

Every person has a persistent ID and seed. Their identity does not reroll after reload.

## Memory

Important choices create memories containing:

- date;
- summary;
- relationship impact;
- importance.

Only the latest twelve memories per NPC are kept in the active save.

## Events

Events are generated from current state, not a random global deck. Current triggers include:

- low coach trust or a bad day;
- depth-chart competition;
- academic risk;
- a teammate's preparation request;
- post-game media attention;
- scheduled follow-ups to previous promises.

An unresolved situation must be handled before the next day or match. Event types have cooldowns so the same conflict cannot repeat every day.
