# PROSPECT — Game Bible

## Первая играбельная цель

Игрок создаёт школьного спортсмена, проходит последние недели сезона, планирует нагрузку, учится, взаимодействует с командой, участвует в ключевых эпизодах матчей, получает интерес колледжей и принимает первое серьёзное карьерное решение.

## Текущий срез

В активной разработке находится только American Football. Basketball не загружается и не создаёт состояние.

## Главный цикл

Недельный план → ежедневная симуляция → события → тренировки → матч → последствия → рекрутинг → новая неделя.
