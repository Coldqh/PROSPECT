# Архитектура

## Границы

- `core` не зависит от конкретного спорта.
- `sports/football` реализует интерфейс спортивного модуля.
- React отображает состояние и отправляет команды, но не выполняет симуляцию.
- Случайность проходит через `SeededRandom`.
- Сохранения версионируются и проходят проверку схемы.

## Сохранения

IndexedDB содержит:

- индекс карьер;
- основные снимки;
- до пяти резервных автосохранений;
- ручные сохранения;
- настройки;
- журнал миграций.

## GitHub Pages

Используется `HashRouter`, относительный `base` Vite и PWA-кэш. Серверная маршрутизация не требуется.
